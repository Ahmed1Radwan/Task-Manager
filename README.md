# Task Manager: is a simple Back-Ebd Localhost Server
> Tools Used 
- Express framework
- Nodejs
- MongoDB

> Description
#### Creating REST API Doing the basic database Methods 
- Get Task
- Create New Task (Post Request) 
- Get All Tasks
